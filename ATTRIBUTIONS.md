## License Notes

- **MIT-licensed dependencies** require preserving copyright notices and license text; upstream `LICENSE` files ship inside `node_modules/` when installed.
- **Apache-2.0 dependencies** (e.g. Prisma) require preserving copyright, license text, and any `NOTICE` files.
- **SIL OFL fonts** (e.g. DM Sans) may be bundled and embedded, but the reserved font name and license must be preserved when redistributing.
- **GPL-3.0** applies to the ITAKAI source code itself — see [`LICENSE`](./LICENSE) for the full text.
